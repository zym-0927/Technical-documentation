react原生只能父子传值，无法兄弟传值（当然它全家桶也有专业的传值管理）



父 -> 子(核心：使用自定义属性props来进行传递)
1. 先在子元素身上用自定义属性绑定要传入的值
	<子元素 name={ 要传给儿子的值 } />
2. 在子元素内，使用this.props.属性名 获取自定义属性的值！
	子元素.js
	this.props === 一次性拿到身上绑定的所有自定义属性
	this.props.name //直接就能获取到传入的自定义属性的值


子 -> 父(父元素定义函数，子元素触发此函数并调用传参)
1. 现在父元素身上定义一个接受参数的函数
	父元素.js
	getData(){

	}
2. 把此函数用自定义属性绑定给子元素，让子元素能拿到此函数
	<儿子 fuc={ this.函数名 } />
3. 在子元素中拿到传入的props函数，触发，并传参
	子元素.js
	
	//在指定位置
	this.props.fuc( 传入参数 ) //调用传入过来的父元素函数，传参，完成子->父
